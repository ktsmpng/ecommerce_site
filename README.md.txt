Installation and Environment Setup
1. Install Python 3.6.0
2. navigate to root directory of project 'cd myshop'
3. 'pip install -r requirements.txt'

Running application 
1. navigate to root directory of project 'cd myshop'
2. to run application 'python manage.py runserver'
3. In browser navigate app using default address 'http://127.0.0.1:8000'

Accessing Admin
1. 'http://127.0.0.1:8000/admin'
